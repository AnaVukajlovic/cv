Izdvojeni projekti

SrbijaManastiri – Desktop aplikacija (C#)

Edukativna desktop aplikacija za prikaz pravoslavnih manastira u Srbiji, razvijena u WinForms okruženju sa bazom podataka i administracijom sadržaja.

  • Autentifikacija i korisničke uloge (Admin/Korisnik)
  • CRUD operacije nad sadržajem aplikacije
  • Rad sa bazom kroz ADO.NET i SQL Server LocalDB
  • Galerija, priče, kalendar, kviz i pretraga sadržaja

Tehnologije: C#, .NET Framework, WinForms, SQL Server (LocalDB), ADO.NET

Pravoslavni Svetionik – Master rad (Laravel + Ollama)

Full-stack web aplikacija posvećena pravoslavnim manastirima u Srbiji, Nemanjićima kao najvećim zadužbinarima i istoriji Srbije i SPC, razvijena kao master rad. Projekat je kreiran uz pomoć AI alata kao deo pedagoškog zahteva, sa ciljem demonstracije implementacije lokalnih jezičkih modela u arhitekturu web sistema.

Napomena: Zbog tehničkih ograničenja besplatnog hostinga, trenutna live verzija aplikacije ne koristi AI u realnom vremenu, već podatke uspešno povlači direktno iz relacione baze podataka.

  • Laravel backend arhitektura i MVC šablon
  • Implementacija i upravljanje relacionom bazom podataka
  • Integracija lokalnog Ollama LLM modela za obradu podataka tokom razvoja
  • REST API komunikacija i dinamički frontend

Tehnologije: PHP, Laravel, Blade, MySQL/SQLite, JavaScript, Ollama
