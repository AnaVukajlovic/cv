Projekti

Recepti – Web aplikacija za posna jela

Web aplikacija namenjena jednostavnom pronalaženju i organizaciji recepata za period posta. Fokus je na preglednoj navigaciji, responzivnom dizajnu i brzom pristupu sadržaju.

Funkcionalnosti: pretraga i filtriranje recepata, kategorije jela, kalendar posta, nedeljni planer obroka, kontakt forma.

Tehnologije: HTML, CSS, JavaScript, Bootstrap.

Beauty & Style – Web aplikacija za zakazivanje termina

Responzivna web aplikacija za pregled usluga i zakazivanje kozmetičkih tretmana. Dizajn je fokusiran na čist UI, jasnu strukturu i dobro korisničko iskustvo.

Funkcionalnosti: prikaz usluga i cenovnika, forma za kontakt, sekcija sa informacijama, prilagođen prikaz za mobilne uređaje.

Tehnologije: HTML, CSS, JavaScript.

SrbijaManastiri – Desktop aplikacija (C#)

Desktop aplikacija razvijena u WinForms okruženju sa bazom podataka (SQL Server LocalDB), namenjena edukativnom i informativnom prikazu pravoslavnih manastira u Srbiji. Aplikacija obuhvata module za manastire, ktitore, priče, kalendar i kviz, uz korisničke interakcije.

Implementirano: autentifikacija i uloge (Admin/Korisnik), kompletne CRUD operacije, rad sa podacima kroz ADO.NET, rad sa multimedijom (galerija).

Tehnologije: C# (.NET Framework), WinForms, SQL Server (LocalDB), ADO.NET, Visual Studio.

IoT robot pas

Projekat razvoja interaktivnog IoT robota zasnovanog na Raspberry Pi platformi, sa implementacijom detekcije i prepoznavanja lica u realnom vremenu. Sistem je osmišljen da prepozna vlasnika i reaguje na njegovo prisustvo kroz unapred definisane komande i pokrete.

Implementirano: obrada video signala sa kamere, detekcija i prepoznavanje lica, komunikacija između hardverskih komponenti, izvršavanje Python skripti za kontrolu ponašanja robota, kao i osnovna IoT komunikacija.

Tehnologije: Raspberry Pi, Python, IoT koncepti, embedded sistemi.

Sistem za upravljanje rekreativnim tenisom

Web aplikacija razvijena kao timski projekat, namenjena organizaciji i upravljanju rekreativnim teniskim aktivnostima. Sistem omogućava jednostavno zakazivanje mečeva, evidenciju rezultata i praćenje rang liste igrača.

Implementirano: korisnička registracija i autentifikacija, upravljanje sesijama, unos i prikaz rezultata, administracija korisničkih profila, kao i organizacija mečeva kroz MVC arhitekturu radi jasne podele odgovornosti.

Tehnologije: PHP (MVC arhitektura), JavaScript, MySQL, rad sa relacijama u bazi podataka.

Planovi

NemanjiciWebApp – Laravel + HTML/CSS/JavaScript + Ollama LLM

Web aplikacija posvećena istoriji dinastije Nemanjića i srpskim manastirima, razvijena kao full-stack projekat sa Laravel backend arhitekturom. Sistem omogućava prikaz vladara, hronološku organizaciju sadržaja, detaljne stranice sa istorijskim podacima, kao i pretragu i filtriranje manastira.

Implementirana je relacijska baza podataka sa modelima i migracijama, REST API komunikacija između backend i frontend sloja, kao i modularna struktura aplikacije (MVC arhitektura).

Poseban deo projekta predstavlja integracija lokalnog LLM modela putem Ollama servisa, koji omogućava AI pretragu i generisanje odgovora na osnovu postojećih podataka aplikacije. Fokus je na kontroli odgovora modela i smanjenju halucinacija kroz strukturiran prompt dizajn i validaciju podataka.

Tehnologije: Laravel (PHP), Blade, HTML, CSS, JavaScript, MySQL/SQLite, REST API, Ollama LLM integracija.
